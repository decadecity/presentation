Specificity

.error {
  color: red;
}

#login-form .error {
  color: orange;
}
It turns out there's an ID of `login-form` wrapping the fields so we can use that to target just the errors in that form: What's happening here is that the second declaration uses a more specific selector so any error contained within that form will be `orange` whilst the rest will remain `red`.
