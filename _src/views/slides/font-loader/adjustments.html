Adjustments

body {
  font-family: System-Font;
  letter-spacing: -0.015em;
}

.wf-customfont-n4-active body {
  font-family: Custom-Font, System-Font;
  letter-spacing: normal;
}

One common problem you'll get when you use a font loader is that the custom fonts will be a different size to the system fonts. If this is the case the same amount of text will take up a different amount of space so when the fonts are switched over you'll get some layout reflow. To counter this you can use the class hook to adjust the size and spacing of the system fonts to match the custom font. This way the text will take up the same amount of space and you won't get layout reflows.
