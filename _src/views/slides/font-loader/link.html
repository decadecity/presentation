Linking the CSS to the JavaScript

CSS

fonts.css
font-family: 'Custom-Font';

JavaScript

webfont.js
families: ['Custom-Font']
Now this bit is the magic spell we use to get the CSS and the JavaScript working together. We need to match these two strings. This is how the web font loader knows which fonts to manage. Any custom font families you have declared in the CSS need to be added to this array in the loader.
