document.querySelectorAll('element#id.class') FTW! This is just like jQuery's `$('element#id.class')` CSS selector. Actually, jQuery uses `querySelectorAll` under the hood so insubstantiating all of jQuery just as a wrapper for a native function like this is a huge waste. yeah, sure jQuery extends the selectors you can use.
