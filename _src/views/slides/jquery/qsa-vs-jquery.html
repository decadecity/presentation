Let's use `querySelectorAll` to select a number of elements and perform an operation on each one. First, let's look at the bloated jQuery way: >>>> Now for some top-gun vanilla JS skillz. We're going to use `querySelectorAll` to select our objects. `querySelectorAll` returns a NodeList and we know we're in Chrome so we can just iterate over arrays using the native `forEach`, right? >>>> Well, no. That would be too easy. What `querySelectorAll` returns is an array like object that doesn't have array in its prototype chain. JavaScript - this is why people pick on you. You're returning an array like object with none of the array methods available. Go home, you're drunk. >>>> So what we need to do is use the `forEach` method from the array prototype: Now we can efficiently select DOM elements with some lightweight and easy to understand code we can move on to event handlers.
