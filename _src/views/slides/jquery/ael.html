Event Handling

jQuery

element.on('click', function (event) {
  // Do code.
});

Vanilla JS

element.addEventListener('click', function (event) {
  // Do code.
});
First off, the old jQuery way: >>>> In vanilla JS we have a method called `element.addEventListener` and this works just like the jQuery event handler. Actually, like `querySelectorAll`, this is actually what jQuery is using behind the scenes. So why are we loading in all of jQuery if all we're using is event handlers?
